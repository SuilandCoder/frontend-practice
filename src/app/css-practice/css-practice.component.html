  • 定位：

    absolute: 脱离原来位置进行定位(其他控件会占据该控件原来位置)，相对于最近的有定位的父级进行定位，没有则相对于文档定位！

    relative: 保留原来位置进行定位，相对于自己原来的位置进行定位！

    定位技巧: relative作为参照物，absolute用来定位（不会因为父级位置变动导致定位错乱）。
